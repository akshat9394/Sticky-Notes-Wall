# 🚀 Sticky Notes Wall Pro

🚀 Built for Web Dev Club Recruitment — designed with focus on **clean UI, smooth UX, and efficient frontend architecture using core web technologies**.

A modern sticky notes web app that allows users to quickly capture ideas, organize them visually, and interact with notes dynamically — all built using **pure HTML, CSS, and JavaScript**.

## ✨ Features

* 📝 Create, edit, and delete notes
* 🎨 Color-coded notes for better organization
* 📌 Pin/unpin important notes
* 🔍 Real-time search & filtering
* 💾 Persistent storage using localStorage
* 🌙 Dark mode toggle
* 🧲 Drag & drop note reordering
* 📱 Fully responsive design
* ✨ Smooth CSS animations & micro-interactions

---

## ⚙️ Run Locally

Clone the repository:

```bash

cd sticky-notes-wall
```

Open the project:

* Simply open `index.html` in your browser
  **or**
* Use Live Server in VS Code for better experience
